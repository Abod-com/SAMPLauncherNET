## Description
CSL (Community San Andreas Multiplayer Launcher) is a launcher for San Andreas Multiplayer

## Why not the native San Andreas Multiplayer launcher?
CSL is a more modern approach to the default San Andreas Multiplayer launcher, which contains more features.
